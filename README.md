# AllroundPersonalScheduleAssistance
APSA is a browser-based task management and time tracking system built with vanilla JavaScript. It features a hierarchical task tree, recurring task reset, progress tracking, scheduling, a precision timer, and a gamified XP system. It includes Kanban, daily review, and analytics modules.
## 项目简介

APSA（全方位个人日程助手）是一款功能完善的个人任务管理与时间追踪系统。系统采用任务树架构，支持复杂项目的任务分解、依赖管理和分类标签。核心功能包括日常任务自动重置、普通任务进度追踪、日程与事件管理、精确计时器、游戏化经验值激励系统等。系统提供看板、管理、复盘三大模块，其中复盘模块包含365天日历视图的每日复盘功能，支持记录每日进展和评分，并通过颜色编码直观展示整年表现。系统支持文件系统存储和LocalStorage双重存储方式，确保数据安全可靠。此外还包含中期目标管理、留言板、能力雷达图、时间统计等辅助功能，帮助用户全面掌握个人时间管理和任务执行情况。系统采用原生JavaScript开发，无需后端服务器，可直接在浏览器中运行。

## 功能特性

### 核心功能
- ✅ **任务树系统**：支持复杂项目的任务分解和依赖管理
- ✅ **游戏化激励**：基于投入时间和难度的经验值系统
- ✅ **多维度任务管理**：日常任务、普通任务、事件三种类型
- ✅ **时间追踪**：精确的任务计时功能
- ✅ **数据可视化**：能力雷达图、统计图表
- ✅ **日程与事件管理**：日程安排和近期事件提醒

### 主要页面

2. **任务管理 (Tasks)**
   - 任务树管理

3. **统计报告 (Reports)**

## 快速开始

### 使用方法

1. 直接在浏览器中打开 `index.html` 文件
2. 系统会自动初始化，数据存储在浏览器的 LocalStorage 中
3. 开始创建任务树并管理你的任务

### 基本流程

1. **创建任务树**
   - 进入"管理"页面 → "任务树"标签
   - 点击"创建任务树"
   - 填写任务树信息并添加子任务

2. **执行任务**
   - 在"看板"页面查看待办任务
   - 点击"开始任务"按钮
   - 使用计时器追踪时间（普通任务）或直接完成（日常任务/事件）

3. **查看统计**
   - 进入"统计"页面
   - 查看能力雷达图、时间统计、完成统计等

## 数据模型

### 任务树 (Task Tree)
- 包含多个子任务的项目
- 支持状态：进行中/已完成
- 完成后可进行复盘和评分

### 子任务 (Subtask)
- 类型：日常任务/普通任务/事件
- 难度：铜/银/金
- 支持依赖关系
- 支持重复设置（日常任务）

### 经验值系统
- 计算公式：`经验值 = 投入小时数 × 难度系数 × 专注系数`
- 难度系数：铜 1.0 / 银 1.5 / 金 2.2
- 固定升级：每级需要 100 XP（可配置）

## 数据管理

### 导出数据
- 进入"系统设置"页面
- 点击"导出数据 (JSON)"
- 数据会以 JSON 格式下载

### 导入数据
- 进入"系统设置"页面
- 点击"导入数据 (JSON)"
- 选择之前导出的 JSON 文件

### 迁移旧数据
- 如果系统检测到旧格式的数据，可以使用"迁移旧数据"功能
- 系统会自动将旧数据转换为任务树格式

## 浏览器兼容性

- Chrome/Edge (推荐)
- Firefox
- Safari
- 需要支持 ES6+ 和 LocalStorage

## 注意事项

1. **数据存储**：所有数据存储在浏览器本地，清除浏览器数据会丢失所有记录
2. **数据备份**：建议定期导出数据备份
3. **浏览器兼容**：建议使用现代浏览器以获得最佳体验
